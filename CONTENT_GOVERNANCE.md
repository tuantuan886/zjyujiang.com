# 宇匠官网内容治理

## 内容原则

1. 企业身份、电话、邮箱、产品规格和测试文件仅使用企业提供的一手资料。
2. 规格、材料和性能结论必须带适用条件，不把单一型号或报告扩展到全部产品。
3. 未提供证书编号的认证、专利、荣誉和工商字段不公开展示。
4. 客户名称与案例仅在获得公开授权并可核验后发布。
5. 正式采购与安全关键连接以图纸、技术协议、批次检验和检测报告为准。

## GEO 与 EEAT 实现

- 每个核心主题都有独立、可索引的语义页面。
- 页面先给直接答案，再解释原理、边界和证据。
- 企业、产品、FAQ、文章、联系页和面包屑使用 JSON-LD 结构化数据。
- 技术文章标注作者组织、发布日期、修改日期、来源和适用边界。
- 企业名称、电话、邮箱、产品名和规格口径在全站保持一致。
- 图片使用真实产品素材，并提供描述性替代文本。
- 提供 `robots.txt`、站点地图生成脚本、语义化 HTML 和补充性的 `llms.txt`。

## 更新清单

- 企业注册地址与统一社会信用代码
- 成立日期、注册资本和企业规模
- 正式官网域名
- 质量体系证书、产品认证、专利和荣誉编号
- 获得授权的客户案例
- 最新产品图纸、检测报告版本和公开范围
